 EDA Retail Project

 Project Overview
This project performs Exploratory Data Analysis (EDA) on a retail sales dataset to understand sales trends, category performance, profit and loss, and customer behavior.

  Dataset
- File: `retail_sales.csv`
- Description: Contains retail sales data including year, category, sales, quantity, discount, and profit.

 Tools & Libraries Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

 Analysis Performed
- Data cleaning and preprocessing
- Year-wise sales analysis
- Category-wise sales comparison
- Profit and loss analysis
- Correlation heatmap
- Data visualization using bar and line charts

 Key Insights
- Electronics category shows highest sales
- Seasonal trends observed in yearly sales
- Discounts impact profit margins

 How to Run
1. Open the notebook in Google Colab
2. Upload `retail_sales.csv`
3. Run all cells to reproduce the analysis

 Conclusion
This EDA helps understand retail sales behavior and supports better business decision-making.
